Block All Ads On Apps and Websites on Rooted Android

Got irritated because of the ads? You can easily get rid of these ads. The common way of doing this is by using an Adblocker. Adblocker is a program that blocks all advertisements on your webpage or app. The popular Adblocker to this is AdAway adblocker. But to use AdAway, you need to have a rooted Android Device. If your device is not rooted, see how to block ads on non-rooted android.


How does AdAway adblocker work?

AdAway is an open-source adblocker for Android that uses the hosts files to block ads. The hosts file contains a list of mappings between hostnames and IP addresses. When an app requests an ad, that request is directed to 127.0.0.1 which does not return any ads and as a result, you won't see any ads.

Advantages of using Adblocker

  1. Increases your browsing experience.
  2. Get rid of the annoying ads.
  3. Prevents the collection of your personal data by Ad Servers.
  4. Reduce your data consumption and many more.

Limitations of using Adblocker

  1. The ad revenue of a developer gets reduced.
  2. You won't be able to access some apps and websites that use adblocker detection.
  3. You wont't be able to get rewards or unlock a feature in games or apps by watching video ad after using an Adblocker.

Steps to block ads using AdAway

  1. First of all, you need to download and install AdAway in your Android. Click here to download AdAway Adlocker.
  2. After downloading, install the app.
    Make sure to enable installing apps from Unknown Sources before installing the app.
  3. After installation is complete, open the app.
    It will ask for root permission. Click on Allow.
  4. After granting the root permission, you can see window like this.
  5. Click on DOWNLOAD FILES AND APPLY AD BLOCKING button.
    You need to have an active internet connection to download host files.
  6. The host files will be downloaded and applied shortly.
  7. After applying the host files successfully, you need to restart your device to make Ad blocking work.
  8. After rebooting your device, you won't see any ads on your mobile app or webpages.

Disabling Adblocker

To disable ad blocking, click DISABLE AD BLOCKING button on AdAway. Doing this will replace the hosts file with the default hosts file of your system. Adblocking will be disabled after rebooting your device and you can see ads again on websites and apps.


Conclusion

Following this tutorial, you have learnt how to block ads on your rooted Android device using AdAway adblocker and also turning off ad blocker.
